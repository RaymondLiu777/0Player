An Unoffical Semi-interactive Recreation of 0PLAYER

 The original game was just a static image and meant to be played in a image editor. This is just a browser-based recreation that mimics many of the features you'd want in an image editor.

Implemented:

  • Draggable tiles
  • Toggleable wires
  • Block grouping
  • Snap to grid for blocks (can be disabled in settings)

NOT Implemented:

  • Any implementation of game mechanics
  • The original art* (some details may be missing)
  • Some late game discoveries that would be too easy to stumble across if implemented

Controls for the website (not the game)

  • WASD/Drag Scroll Wheel: Move camera
  • QE/Scroll Wheel: Zoom in and zoom out
  • Left Click + Drag: Move blocks
  • Right Click/Hold: Toggle sprite
  • Hold G + Left Click: Group blocks
  • Hold G + Right Click: Ungroup blocks
